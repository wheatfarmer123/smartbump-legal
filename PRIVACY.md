# SmartBump Privacy Policy

**Last updated: May 22, 2026**

## 1. Introduction
This Privacy Policy explains what data SmartBump collects, how it is used, and how it is stored. By using SmartBump, you agree to the collection and use of data as described here.

## 2. Data We Collect
SmartBump collects and stores the following data:

**Server Data:**
- Server ID
- Bump channel ID
- Server description and tags
- Invite link
- Bump count and last bump timestamp
- Reminder settings

**User Data:**
- User ID of the last person to bump a server
- User ID and rating score when a server is rated

## 3. How We Use Your Data
The data collected is used solely to operate the SmartBump service, including:
- Broadcasting server advertisements across the network
- Enforcing bump cooldowns
- Sending bump reminders
- Displaying server ratings and statistics

## 4. Data Storage
All data is stored in a private SQLite database. We do not share, sell, or distribute your data to any third parties.

## 5. Data Retention
- Server data is retained for as long as SmartBump remains in your server
- When SmartBump is removed from a server, all associated server data is automatically deleted
- Rating data associated with a server is deleted when that server resets or is removed from the network

## 6. Your Rights
You have the right to request deletion of your server's data at any time by:
- Using the reset option in `/setup`
- Removing SmartBump from your server

## 7. Third Parties
SmartBump operates on Discord's platform and is subject to Discord's own Privacy Policy. We do not share data with any other third parties.

## 8. Changes to This Policy
We reserve the right to update this Privacy Policy at any time. Continued use of SmartBump after changes constitutes acceptance of the updated policy.

## 9. Contact
For privacy concerns or data deletion requests, please join our support server (link coming soon).
